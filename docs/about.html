About This Project

Created for the Purdue Hello World Hackathon by Adam Mirmina, Bonnie Le, and Kyler Yun.

Dates: Sep 20–21, 2025

Last updated: Sep 21, 2025

Study Companion is a lightweight, client-first web app for focused study sessions, matching with study buddies, finding study spots on campus, and private messaging (opt-in; available when your profile is public). It runs entirely on GitHub Pages with Firebase for authentication and data.
